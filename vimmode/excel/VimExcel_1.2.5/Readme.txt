1. VimExcelとは

VimExcelとはExcel上の作業を可能な限り、Vimエディタのキーボード操作にて実現させることを目的としたアドインソフトウェアです。

2. ライセンス

本アドインは、これまでVimxlsとして、公開させていただいたものを改善したものとなります。
ライセンスの基準はクリエイティブコモンズの「表示（CC BY）」相当とお考えいただければと思います。

3. 動作環境および注意事項

作者が利用しているキーボードは、主に英語キーボードです。ちなみに端末日本語キーボードしかない
ないキーボードについては、ドライバを変更して101配列に変更しています。

VBAで利用している関数の変更制限可と思いますが、ツール類でのキー配列の変更では、一部うまく
機能しないものがあります。対応策としては、いまのところ101キー配列のキーボードをご利用いただくか、
本製品で定義しているキー配列を無視して、ご利用いただくほかありません。

Excel2016での動作は確認しておりますが、Excelのバージョンの差異によって
若干挙動が異なる場合があります。
(Excel2003では、正常に動作しない場合があります。)

現行の開発は、Excel2016(32bit)を主体としており、Windows 10(64bit)での稼働を確認しております。

しかしながら、本バージョンについては、Windows10(64bit)上のExcel2016(32bit)の環境においての
再現性の高い不具合も確認されています。

具体的には、下記4.以降でのインストール後の作業をWindowsの再起動を実施するたびに実施しなければ
VimExcelが稼働しないというものです。

高い再現性で、現象は確認できているものの、その根本原因についてはいまだ特定できておらず、
本リリースで本不具合の解消は行われていない点について、ご了承ください。

本不具合の原因、解消法等ご存知の方がおられるようでしたら、下記7.の連絡先のいずれかにご連絡いただければ
幸いです。

4. インストール方法

- インストール方法(Excel 2016)

Developer -> Excel Add-ins ->　Browse -> Vimexcelを選択 -> [OK]

上記Browseで表示されたAddInsフォルダに、zip解凍後に
VimExcel__(バージョン名).xlaを配置してください。

5. 起動・終了方法

Developer -> Excel Add-ins ->　Add-ins available:

インストールしたVimExcelの左側の□をチェックを外して、OK。

[Ctrl + M]で起動となります。
（ブックオープン時に自動起動する仕様となっています。上記不具合のため、正常に起動しない場合が多いです。）

同じく[Ctrl + M]で終了です。

6. ヘルプ

VimExcel起動後に、?により、簡便なヘルプが参照可能となっています。
また、ばしゅらさん - https://twitter.com/basyura - に作成いただいた資料を元に
PowerPointによるヘルプ資料を作成いたしました。
またExcelファイルには、ほぼすべてのコマンドが記載されています。

7. アンインストール方法(Excel 2013)

Developer -> Excel Add-ins ->　Browse

上記で表示される"Add-ins available:"のVimExcel_(バージョン名)のチェックを外し、
"Browse"で表示されたAddInsフォルダから、VimExcel_(バージョン名).vbaを削除してください。
(Excelが起動しており、削除できない場合には、Excel終了後にAddInsフォルダから削除してください。)
